=====================
Using ``add_integer``
=====================

First we import ``add_integer`` from the ``0-add_integer`` module:

    >>> add_integer = __import__('0-add_integer').add_integer

Now use it:

Numbers
=======

>>> add_integer(2, 3)
5

add_integer(2,)
100

>>> add_integer(2, 0)
2

>>> add_integer(-2.3, 4.5)
2

>>> add_integer(-0.3, 4)
4
